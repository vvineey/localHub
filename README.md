# budae-Fe

서울10반 B팀 LocalHub 프런트엔드 저장소입니다.

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 폴더 구조

- `src/views`: 페이지 화면
- `src/components`: 공통 컴포넌트
- `src/router`: Vue Router 설정
- `src/data`: 프런트 확인용 목 데이터
- `src/services`: 백엔드 API 연동 레이어

`.env`는 GitLab에 올리지 않습니다. 필요한 환경변수는 `.env.example`을 참고해 로컬에서 생성하세요.
